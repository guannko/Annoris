## О компании Brain Index

**Кто мы:** AI Automation Agency - агентство по автоматизации бизнес-процессов с помощью AI.

### Что мы делаем:

1. **AI Боты для мессенджеров**
   - Telegram боты (основной фокус)
   - WhatsApp боты
   - Discord боты
   - Функции: ответы клиентам 24/7, приём заказов, консультации, AI-ассистенты

2. **Автоматизация workflow**
   - n8n workflows (основной инструмент)
   - Make.com сценарии
   - Интеграции между сервисами (CRM, Google Sheets, API, мессенджеры)

3. **AI интеграции**
   - GPT, Claude, DeepSeek, Groq
   - Анализ данных, генерация контента, распознавание

4. **Аналитические системы**
   - Дашборды, алерты, мониторинг
   - Telegram уведомления

### Бизнес-модель:
- Проектная работа: €500 - €3000+ за проект
- Поддержка после сдачи: от 2 недель бесплатно, далее по контракту

### Наши продукты (portfolio):
- BTC Analyzer v4.1 - Telegram бот с AI анализом
- AI Fitness Coach - бот с Groq Vision для анализа еды
- Make.com Gateway - централизованное управление 50+ сценариями

## АДМИНКА (для Brain Index team)

**Цель:** Управлять ВСЕМИ нашими продуктами и клиентскими проектами из одного места.

### 1. Управление ботами
- [ ] Список ботов (multi-platform: TG/WA/Discord/Insta)
- [ ] Статус каждого бота (online/offline/error)
- [ ] Быстрые действия: restart, stop, view logs
- [ ] Credentials vault (encrypted storage)
- [ ] Conversation logs (история диалогов)
- [ ] User management per bot (ban/unban/search)
- [ ] Broadcast tool (массовые рассылки)

### 2. Управление автоматизациями
- [ ] n8n workflows: список, статус, последний запуск, ошибки
- [ ] Make.com scenarios: через Gateway
- [ ] Логи выполнения
- [ ] Webhook tester (отправить тестовый запрос)
- [ ] Алерты при ошибках

### 3. Управление клиентами
- [ ] Список клиентов
- [ ] Проекты по каждому клиенту
- [ ] Статус проекта (в работе / сдан / на поддержке)
- [ ] Usage tracking (сколько сообщений, запусков)
- [ ] История коммуникации
- [ ] Контракты, оплаты

### 4. Деплойменты / Инфраструктура
- [ ] Northflank сервисы
- [ ] Vercel проекты
- [ ] Railway apps
- [ ] Статусы, логи, ресурсы

### 5. Мониторинг и алерты
- [ ] Общий health dashboard
- [ ] Настройка алертов (Telegram, email)
- [ ] История инцидентов

## Технический стек (планируемый)

**Frontend:**
- Vite + React + TypeScript
- Tailwind CSS
- shadcn/ui (как в лендинге brain-index.com)

**Почему не готовые шаблоны:**
- Уникальная специфика (боты + n8n + Make.com)
- Полный контроль над кодом
- Единый стиль с лендингом

**Интеграции нужны:**
- Telegram Bot API
- n8n API
- Make.com API (через Gateway)
- Northflank/Vercel/Railway APIs

## Исследование рынка (22.12.2025)

### Что нашли:
- TeleAdminPanel, BotAlto - только боты, без workflow
- n8n-multi-tenant - инфраструктура, не UI
- Botsify, Stammer.ai - SaaS с vendor lock-in
- Twenty, Frappe CRM - generic CRM, не про ботов

### Вывод:
Готовых решений под наш стек НЕТ. Строим сами на shadcn/ui.
